# GCT Fleet Office

This is a browser-based fleet management prototype for Grand Central Terminal dispatch operations.

## Open the game

### Option 1: Open locally (quickest)
1. Download the repo files to your device.
2. Open `index.html` in a web browser.

### Option 2: Host it for iPad/Safari
If your device blocks local scripts, host the files so Safari can run them:
1. Go to https://app.netlify.com/drop
2. Upload `index.html`, `style.css`, and `game.js`.
3. Open the live link Netlify provides.

## Controls
- Click a train in the timetable to open the control panel.
- Assign a consist and conductor.
- Open/close doors, then **Buzz & Dispatch** to send the train.
- Toggle **Office View / Track View** to watch departures.
